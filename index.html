NFCash (Group Project)

A cashless payment system using NFC, made for college canteen. Built using Raspberry PI and Arduino. A webpage was built using PHP and MySQL to allow the user to order food on his computer or mobile. Also a simple android app to scan NFC UID was built to show how NFC works.
I built the android app to scan NFC card's UID. I also helped in testing and debugging code for Arduino, Raspberry PI, and the website. I learned about android app building process, responsive web design, how to securely store data such as passwords in a database, and a bit about how Arduino and Raspberry PI work.
